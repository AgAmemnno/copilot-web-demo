汎用フォントファミリー視覚デモ

CSSの汎用フォントファミリーがどのように表示されるかを確認します。
実際の表示はOSやブラウザに依存します。

Serif (セリフ体) - serif

特質: 文字のストロークの端に「セリフ」と呼ばれる小さな飾りがあるフォントです。 伝統的、公式、読み物などの印象を与えやすく、長文の可読性が高いとされることがあります。 代表例: Times New Roman, Georgia, MS P明朝, 游明朝

Sans-serif (サンセリフ体) - sans-serif

特質: 「サン (sans)」はフランス語で「〜なし」を意味し、セリフがないフォントです。 モダン、シンプル、クリーンな印象を与えやすく、ウェブサイトの本文や見出し、UI要素など幅広く利用されます。 代表例: Arial, Helvetica, Verdana, MS Pゴシック, ヒラギノ角ゴ, メイリオ

Monospace (等幅フォント) - monospace

特質: すべての文字が同じ幅（等幅）で表示されるフォントです。 主にプログラミングのソースコードや、テキストエディタ、ターミナル表示など、文字の位置を正確に揃えたい場合に使用されます。 代表例: Courier New, Consolas, Osaka−等幅, ＭＳ ゴシック (一部)

Cursive (筆記体フォント) - cursive

特質: 手書きのような流れるような筆跡を持つフォントです。abcdefg エレガント、フォーマル、またはカジュアルな手書き感を表現するのに使われます。 ただし、英語圏の筆記体を想定しており、日本語環境で適切なフォントが適用されることは少ないです。 代表例: Comic Sans MS (広い意味で), Zapf Chancery, Brush Script MT

（注: 日本語環境では適切な筆記体フォントが適用されない場合があります。）

Fantasy (装飾フォント) - fantasy

特質: abcdeABCDE 通常のフォントとは異なり、非常に装飾的で、特定のテーマや雰囲気（例: ファンタジー、SF、レトロなど）を強く打ち出すためのフォントです。 本文には不向きで、主にタイトルやロゴなど、デザインのアクセントとして使用されます。 日本語環境では適切なフォントが適用されることは稀です。 代表例: Impact, Papyrus, Cooper Black

（注: 日本語環境では適切な装飾フォントが適用されない場合があります。）

System UI - system-ui

特質: ユーザーが使用しているオペレーティングシステム（OS）のデフォルトユーザーインターフェース（UI）フォントを使用します。 これにより、ウェブサイトやアプリケーションがOSのネイティブな見た目と調和し、ユーザーに親しみやすい体験を提供できます。 代表例: WindowsではSegoe UI、macOS/iOSではSan Francisco、AndroidではRobotoなど、OSによって異なります。

UI Serif - ui-serif

特質: OSのデフォルトUIフォントのうち、セリフ体のフォントを使用します。 system-uiがサンセリフ体の場合に、明示的にセリフ体を要求する際に使えます。

UI Sans-serif - ui-sans-serif

特質: OSのデフォルトUIフォントのうち、サンセリフ体のフォントを使用します。 system-uiとほぼ同等か、より明示的にサンセリフを指定する場合に使います。

UI Monospace - ui-monospace

特質: OSのデフォルトUIフォントのうち、等幅フォントを使用します。

UI Rounded - ui-rounded

特質: OSのデフォルトUIフォントのうち、角が丸いデザインのフォントを使用します。

（注: OSやブラウザが対応する角丸フォントを持っている場合に効果が見られます。）

Math - math

これはMathフォントのサンプルです。∑ ∫ √ ± ∞ (x + y)² = x² + 2xy + y²。数式用。 特質: 数学的な記号や数式を表現するために特別に設計されたグリフを含むフォントです。MathMLなどの数式表示に適しています。

（注: 特定の数式文字やMathMLと組み合わせることで効果が顕著になります。）

Emoji - emoji

これはEmojiフォントのサンプルです。😀🎉👍💯 あいうえお。 特質: 絵文字を表示するためのフォントです。カラー絵文字などをサポートします。

Fangsong (仿宋体) - fangsong

これは仿宋体のサンプルです。The quick brown fox jumps over the lazy dog. 12345. あいうえお。你好世界。 特質: 中国語のフォントスタイルの一つで、宋朝時代の木版印刷のスタイルに由来します。 楷書体と宋体（明朝体）の中間のような特徴を持ち、やや硬筆で書いたような、整然としつつも流麗さのある書体です。 主に中国語の文脈/で使用されますが、日本語の漢字表示にも影響する場合があります。

（注: 主に中国語フォントとして利用可能か、日本語では明朝体に近い表示になることがあります。）
