説明:

  • 全体のレイヤー順序 (main.css): reset, framework, project-specific, utilities。
  • framework レイヤー:
    • imported-styles.css をインポートします。
    • imported-styles.css 内のレイヤー base と theme は、この framework レイヤーの一部として、宣言順 (base → theme) に組み込まれます。
    • imported-styles.css 内のレイヤー化されていないスタイル (p.imported-unlayered-text) は、framework レイヤー内で、base と theme の後に続きます。
    • main.css の framework レイヤーで直接定義されたスタイル (.component-a) も、このレイヤーに属します。
  • .action-button のスタイルカスケード:
    1. framework.base (imported): background-color: #ecf0f1;
    2. framework.theme (imported): background-color: #f1c40f; (baseを上書き)
    3. project-specific (main): background-color: #c0392b !important; (themeを上書き、!importantも影響)
  • .component-b のスタイルカスケード:
    1. framework.base (imported): background-color: #fef5e7;
    2. framework.theme (imported): background-color: #fff9c4; (baseを上書き)
